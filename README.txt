BẢN FIX - Host Cầu Lông

Cách chạy:
1. Giải nén file zip.
2. Mở index.html bằng Chrome / Edge / Safari.
3. Dữ liệu lưu bằng localStorage trên đúng trình duyệt đang dùng.

Nếu Chrome báo cảnh báo file:// trong Console:
- Nếu app vẫn chạy thì bỏ qua.
- Nếu muốn sạch Console hơn, chạy bằng server local:
  Windows: mở CMD trong thư mục này rồi chạy:
  python -m http.server 8000
  Sau đó mở: http://localhost:8000

Lưu ý:
- Đừng xoá cache nếu chưa xuất JSON.
- Tab Dữ liệu có nút Xuất/Nhập JSON để backup.
